## Hello! Welcome to our Database Management Program!


## Table of Contents: File Directory

- `lib`: the folder to maintain dependencies with our squlite jar file
    - `lib\sqlite-jdbc-3.51.0.0.jar`: our imported sqlite library
- `checkpoint_four.db`: our imported database from squlite used for our program
- `DatabaseControl.class`: Assissts DatabaseControl.java
- `DatabaseControl.java`: Connects to the database and does every call for selecting, updating, and deleting data
- `InfoSystem.class`: Assissts InfoSystem.java
- `InfoSystem.java`: Methods for traversing menus
- `ReportGenerator.java`: Creates reports
